Welcome to the Documentation Hub

Terror in Tierhaven plunges your party into a nightmare scenario where malevolent monsters and rival factions clash in a dark city devastated by a mysterious meteor shower. This comprehensive documentation will help you get the most out of your Foundry VTT module.

Current Version: 1.0.7 | Foundry Compatibility: v13+ | D&D 5e System: 5.0.0+

Quick Start Guide

Installation

  1. Open Foundry VTT and navigate to the setup page
  2. Click "Add-on Modules" ‚Üí "Install Module"
  3. Search for "Sebastian Crowes Terror in Tierhaven"
  4. Click "Install" and wait for completion
  5. Enable the module in your world's module settings

First Time Setup

  1. Import the adventure from the compendium
  2. Review the Table of Contents journal entry
  3. Configure world time for daily monster spawning
  4. Set up player permissions for journal entries
  5. Test scene transitions and teleport zones

Quick Troubleshooting

Common issues and quick fixes:

  • Module not appearing: Check Foundry v13+ compatibility
  • Broken formatting: Clear browser cache (Ctrl+F5)
  • Monsters not spawning: Enable world time tracking
  • Missing images: Check internet connection for GitHub assets
